# Serverless Youtube Video Extractor

[Demo](https://youtube-extractor.mydrv.workers.dev/)

## Usage

```
/?id={videoID}

e.g. https://your-domain/?id=dQw4w9WgXcQ
```

## Development

```
npm install

npm run start
```
